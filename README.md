# MAXSS CLI

Repository:
- https://github.com/soslaxx/maxss

For protocol internals and design notes, see [working.md](./working.md).

## Install CLI Version

```bash
git clone https://github.com/soslaxx/maxss
cd maxss/maxss-core
chmod +x install.sh
sudo ./install.sh
```

## First Run

```bash
maxss
```

In the menu:
- choose `9` (`Quick secure config`) to create a standard working config quickly;
- then create a user and generate a subscription link.

## Important

I have not tested every feature of this project.
I only tested the standard config created via option `9` in the menu.
